HTML (HyperText Markup Language)

O HTML (Linguagem de Marcação de Hipertexto) é a base da construção de páginas web. Ele é uma linguagem de marcação usada para estruturar e organizar o conteúdo exibido nos navegadores, como textos, imagens, links, vídeos e outros elementos interativos. Embora o HTML sozinho não seja uma linguagem de programação, ele trabalha em conjunto com outras tecnologias como CSS (para estilizar) e JavaScript (para interatividade), formando o núcleo do desenvolvimento web. A versão mais recente, HTML5, introduziu novos recursos como suporte nativo a multimídia, gráficos e maior acessibilidade

CSS (Cascading Style Sheets)

O CSS (Folhas de Estilo em Cascata) é uma linguagem usada para estilizar e definir a apresentação visual de páginas web. Ele trabalha em conjunto com o HTML para controlar a aparência dos elementos, como cores, fontes, espaçamentos, tamanhos, bordas, alinhamentos, animações e muito mais. Com o CSS, os desenvolvedores podem criar designs atraentes, organizar layouts responsivos que se adaptam a diferentes dispositivos (como desktops, tablets e smartphones) e melhorar a experiência do usuário. Ele utiliza seletores para identificar quais elementos devem ser estilizados e propriedades para definir o estilo aplicado.

JavaScript (JS)

O JavaScript é uma linguagem de programação de alto nível e interpretada, amplamente utilizada para criar interatividade e dinamismo em páginas web. Diferente do HTML e CSS, que estruturam e estilizam a página, o JavaScript permite adicionar funcionalidades avançadas, como validação de formulários, animações, atualizações em tempo real e manipulação de elementos do DOM (Document Object Model). É uma das três principais tecnologias do desenvolvimento web, funcionando em conjunto com HTML e CSS. Inicialmente usado apenas no lado do cliente (frontend), com o advento de tecnologias como o Node.js, o JavaScript também passou a ser executado no lado do servidor (backend).
